# A completely normal TODO List App repo
Definitely not managed by a [victorian ghost child](https://github.com/pawap90/ghostbot).

## Quick Start

1. Clone the repo
2. Install dependencies

```
npm install
```

3. Start the server

```
npm run dev
```
